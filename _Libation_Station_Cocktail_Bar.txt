# Libation Station Cocktail Bar

## Description

Libation Station is a virtual web page of a cocktail bar designed to present the bar's menu, provide information about the establishment, and allow users to make reservations. The web page includes sections such as Home, About Us, Reservation, Menu, Photo Gallery, and Contact. It offers a wide range of non-alcoholic beverages, beers, wines, and cocktails along with their respective prices. Users can also view the bar's location on a map and get in touch through provided contact information.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Features

- Responsive design: The web page is designed to be accessible on various devices, including computers, tablets, and smartphones.
- Navigation: Users can easily navigate through different sections of the web page using the navigation bar.
- Reservation: Users can reserve a spot at the bar by accessing the reservation section.
- Menu: The web page showcases a comprehensive menu featuring various beverages and cocktails along with their prices.
- Photo Gallery: Users can view images of the bar and its ambiance in the photo gallery section.
- Contact: Users can find the bar's address, phone number, and email address in the contact section.
